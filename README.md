# HomeTask3_PracticeGroup_AntonBukalo
solitaire
Выполнен базовый уровень :
- Размешивание и распределение карт случайным образом 
- Выдача карт из колоды раздачи по 1 
- Колода раздачи зациклена, после выдачи последней карты возможно снова начать выдачу с первой карты 
- Карта выбирается по клику и перемещается тоже кликом 
- Карту или стопку можно переместить на место пустой игровой колоды, если эта карта 
- Король (верхняя карта, для стопки) 
- Двойной клик на карту пытается отправить её в финишную колоду 
- Карту можно переместить только на ту карту, которая выше достоинством на 1 и имеет противоположную масть 
- Можно переместить стопку карт, за карту отсчета для перемещения берется карта в стопке на которой произошел клик 
- Карту можно переместить в финишную колоду, если эта колода
    с картами той же масти и последняя карта в колоде ниже достоинством на 1 или же колода пустая и перемещаемая карта является тузом 
- Если после пемерещения карты в игровой колоде еще остались карты, колода должна открыть нижнюю карту
- Игра считается оконченной, если все карты перемещены в финишные колоды 
- Вывод поздравительного сообщения 


Структура будет изменена после практики (если это нужно), т.к. есть вопросы с Event и как их использовать( к примеру вызвать Card.onClick  в dealDeck для возвращения карты)
